== 커널 ==
>> Common <<
- Google 의 7월 패치들과 Android O DP3 패치들 적용
- Linux 3.10.107
- xpad: 최신 Linux 업스트림 패치들 적용
- kms: 최신 Linux 업스트림 패치들 적용
- net, mm, sdcardfs: Google android-3.10 로 부터 최신 패치들 적용
- msm_thermal: 낮은 온도에서 더 부드러운 성능을 낼 수 있도록 조정!
- msm_thermal: 쓰로틀링 지연 구현
(주파수가 바로 제한되지 않고, 약간의 지연이 된 후에 제한이 됩니다.
 또한, 온도가 내려가면 제한을 해제하고, 지연시간을 초기화 시킵니다.
 따라서, 이 시간동안 성능을 보장합니다. 기본값은 4초입니다.)
- msm_thermal: Little 의 코어 컨트롤 구현
(성능과 온도가 일부 Little 코어들을 끄고, 남아있는 코어들에게
 높은 주파수들을 주는 것이 더 좋습니다.)
- qcacld-2.0(wlan driver): LA.BF64.1.2.3_rb1.14 로 업데이트 (from CAF)

<b><#selectbg_g>[bullhead_II-v23.0]</#></b>
== 커널 ==
>> Common <<
- v22.2 에서 적용된 CAF 의 보안 관련 패치들 취소 (배터리 드레인을 보임.)
- qcacld-2.0(wlan): CAF 의 LA.BF64.1.2.3_rb1.13 드라이버 사용
- msm_thermal: 장치가 시원할 때 'poll_ms' 를 늘림
- msm_thermal: 화면이 꺼졌을 때, 온도를 장치가
 시원해질 때까지 확인한 후 suspend 상태로 들어가도록 함
- msm_thermal: 주파수 제어의 반복되는 행동들 제거
- wcnss: 잠재적인 메모리 누수와 힙 오버플로 수정 (from CAF)
- sdcardfs, tcp, dm, gre: Google android-3.10 으로 부터 최신 패치들 적용

- f2fs-tools: dev 브랜치의 최신 변경 사항들 적용

- Updater: 라이브러리들을 업데이트하고 API 26 으로 재빌드
- A2DPChecker: 라이브러리들을 업데이트하고 API 26 으로 재빌드
