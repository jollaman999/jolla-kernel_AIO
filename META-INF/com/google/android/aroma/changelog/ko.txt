  0 : 노크온 끄기
  1~9 : n+1 번의 탭으로 깨우기. (n = 설정값)

  예시)
  0 : 노크온이 작동하지 않습니다.
  1 : 두번 탭하여 폰을 깨웁니다.
  2 : 세번 탭하여 폰을 깨웁니다.
  9 : 열번 탭하여 폰을 깨웁니다.

  인식 시간 : 100ms * '터치 수'
  (따라서 9로 설정하셨을 경우 1초안에 10번을 두들겨야 폰을 깨울수 있습니다. :p)

- 커널 설정 일부 잘못된 부분들 수정 및 최적화
- 그외 일부 마이너 패치들 적용
-  omni 커널 지원
- 2014/09/10 GEEB 커널 지원
- 2014/09/12 기존 카메라 드라이버 (mako/넥서스4 기반) 가 포함된 버젼들 추가 업로드


<b><#selectbg_g>[v9_Refresh]</#></b>

- 커널 안정화 : Linux 3.4.103 패치를 하면서 소스상에서 잘못된 부분들 수정
- mod/nonmod 커널 동시 지원
- 사용자 측에서 조절 가능한 쓰로틀링 기능적용 (Trickstermod 앱에서 온도및 클럭 단계별 조절가능)
(※ 온도제한 기능은 TricksterMod 앱의 특정 메뉴에서 '온도 제어' 기능을 통해 직접 설정 하실 수 있습니다.)
(※ 온도 제한 기능은 부팅시 기본 클럭값인 1134MHz 에 최적화 되어있습니다. 오버클럭을 하시거나 더 나은 성능을 원하신다면 '온도 제어' 기능 메뉴에서 클럭및 온도를 상향 조정 해주시기 바랍니다.)

<b><#selectbg_g>[v9]</#></b>

- Intelli-plug 제거 (호환성 부족) 및 MSM_HOTPLUG 드라이버 적용 (더 나은 핫플러깅 지원)
- 사용되지 않는 USB 네트워크 드라이버들 비활성화
- ARM RWSEM, AES, SHA1 알고리즘 최적화
- ARM CPU 를 위한 최적화 적용 (Allow CPU-supported unaligned accesses, Manage only online cpus)
- 안드로이드 logcat 버퍼 사이즈 감소 256->128
- 커널 안정화를 위한 전압 및 클럭 부분들 수정
- Intellidemand 거버너 v5.0 으로 업데이트 및 유사 기능을 구현하는 Slim 거버너 제거
- Smartmax 거버너 suspend_ideal_freq 수정 594000 -> 702000
- IO 성능 향상 및 앞서 버퍼 크기 읽기 사이즈 증가 128->512
- L2 캐시 수정으로 인한 전체적인 성능 향상
- 마운트시 기본으로 noatime, nodirtime 옵션을 적용토록 하여 IO 성능 향상
- writeback, sched, SELinux 최적화
- Faux Sound v3.5 로 업데이트
- Linux 3.16.1 기준 F2FS 업데이트
- 노크온 밀어서 깨우기 사용되지 않는 부분 제거
- ZRAM 지원
- 온도제한 및 스로틀링 기능을 jolla-kernel 에 최적화하여 직접 수정 (@myfluxi, @motley 님 소스 참고)

※ 온도제한 기능은 TricksterMod 앱의 특정 메뉴에서 제한 온도, 최소 클럭, 사용 유무를 직접 설정 하실 수 있습니다.
※ 온도 제한 기능은 부팅시 기본 클럭값인 1134MHz 에 최적화 되어있습니다. 오버클럭을 하시거나 더 나은 성능을 원하신다면 '온도 제어' 기능 메뉴에서 클럭및 온도를 상향 조정 해주시기 바랍니다.

※ jolla-kernel 의 모든 기능들은 TricksterMod 앱에서 지원됩니다. 그외의 커널 컨트롤 앱에서는 일부 지원되지 않는 기능들이 있을 수 있습니다.
