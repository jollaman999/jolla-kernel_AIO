<b><#selectbg_g>[bullhead_II-v15.0]</#></b>
== 커널 ==
- 리베이스!! (엄청 부드럽고 거의 0~1% 에 가까운 배터리 드레인!!)
- msm_thermal: vdd 제한에 대한 메세지 출력
- msm8992: vdd 제한 비활성화
- msm_thermal: 일부 파라미터들 이름 변경
- msm_thermal: 빅 클러스터를 개별적으로 조절함 (더 부드러워짐.)
(현재, 쓰로틀링 온도에 도달하면, 리틀과 빅이 동시에 주파수를 감소하게 됩니다.
빅 클러스터를 개별적으로 조절할 수 있도록 하였습니다.
따라서 이로 인해, 더 높은 온도에서 빅클러스터의 주파수를 감소시킬 수 있습니다.)
- msm_thermal: 파라미터들 설명 수정
- touchscreen, mdss: 사운드가 멈춘 후 터치하여 깨우기 기능들이 작동하지 않는 문제 수정.
- touchscreen: synaptics: 화면이 켜져있는 경우 터치스크린을 끄도록 시도하지 않음.
- s2w: s2w_input_event() 에서의 잘못된 체크들 수정
- mdss, touchscreen: touch on 체크들 단순화
- cpufreq: cpu-boost: input_boost_ms 를 '1500' 으로 증가
('msm_performance' 를 포함하여 많은 것들을 비활성화 시키고 최적화를 하였습니다. 그리고 40 은 높은 주파수를 유지하기에 너무 짧습니다.
터치를 하는 동안 높은 주파수를 유지 할 수 있도록 이 값을 증가시켰습니다.)
- tcp: 소켓들이 MSG_FASTOPEN 을 기본으로 사용 할 수 있도록 함.
- ASoC: msm: qdsp6v2: 돌비의 CVE-2016-2469 수정 (CAF 로 부터 가져옴)
- ASoC: msm: qdsp6v2: 비활성화된 모듈들을 컴파일 하도록 복구
- crypto: sha256 - SHA-256 에 대한 베이스 레이어 구현
- crypto: sha512 - SHA-512 에 대한 베이스 레이어 구현
- crypto: arm64/sha2 - OpenSSL 의 SHA256/SHA512 구현 통합
- ANDROID: dm verity: 최소 프리패치 크기 추가
(eMMC 와 같은 장치는, 한번에 더 많은 해시 블럭을 읽는데 더 나은 성능을 보여준다.
안드로이드에 대해서, 128 을 기본값으로 한다. 테스트 된 기기들에서 300ms 의 부팅 시간이 줄었다.)
- 'LineageOS/android_kernel_oneplus_msm8996' 로 부터 일부 CAF 패치들 적용
- CAF 로 부터 카메라 패치들 적용
- cpufreq: Relaxed 거버너 추가
- Makefile: Linaro 최적화 추가
- qcacld-2.0: O3 로 빌드
- qcacld-2.0: maybe-uninitialized 경고 메세지 수정
- ext4: 3.18.y 로 부터 일부 업스트림 패치들 적용
- adredo-idler 제거
- per-process reclaim 제거
- 오래된 CAF 패치들 제거

== AIO ==
- ramdisk: 'input_boost_ms' 라인 제거
(기본값이 커널에 의해 1500 으로 변경됨.)
- ramdisk: devfreq 라인들 제거
- ramdisk: per-process reclaim 라인들 제거
- 마쉬멜로우 지원 제거
